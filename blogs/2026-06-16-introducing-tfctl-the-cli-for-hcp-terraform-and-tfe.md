tfctl is the first dedicated CLI for HCP Terraform and Terraform Enterprise, giving engineers and AI agents comprehensive, secure access to platform APIs. It includes built-in safety features like dry-run previews and delete confirmations, schema discovery, flexible output formats (JSON, markdown, human-readable), and full API coverage based on OpenAPI specifications.
